# 📘 DSA Handwritten Notes — Notebook

Complete 10-chapter Data Structures & Algorithms notes in a handwritten-notebook style, with diagrams, Python code, 30 coding interview questions and 600 practice questions per chapter.

## 🚀 GitHub par Free Host Karna (GitHub Pages)

### Step 1 — Naya Repository Banao
1. GitHub par login karo → top-right **+** icon → **New repository**
2. Repository ka naam do, jaise `dsa-notes`
3. **Public** rakho (Pages free tier ke liye public zaroori hai, agar free account hai)
4. **Create repository** click karo

### Step 2 — Files Upload Karo
Sabse aasaan tarika (bina Git command line ke):
1. Apne naye repo page par jao
2. **"uploading an existing file"** link par click karo (ya **Add file → Upload files**)
3. Is folder ki saari `.html` files (index.html, chapter1.html ... chapter10.html, final_revision.html) ek saath drag-drop karo
4. Neeche **Commit changes** click karo

### Step 3 — GitHub Pages Enable Karo
1. Repo ke andar **Settings** tab par jao
2. Left sidebar mein **Pages** par click karo
3. **Source** ke niche → **Deploy from a branch** select karo
4. Branch = `main`, folder = `/ (root)` select karo → **Save**
5. 1-2 minute wait karo — GitHub ek link degा jaisे:
   ```
   https://<your-username>.github.io/dsa-notes/
   ```

### Step 4 — Check Karo
Us link ko browser mein kholo — `index.html` automatically home page ki tarah khulega, jisse saare chapters aur final revision notebook tak navigate kar sakte ho.

## 💡 Tips
- Sab files ko **isi naam se** rakhna zaroori hai (chapter1.html, chapter2.html, etc.) kyunki index.html inhi naamon se link karta hai.
- Agar aap repo ka naam kuch aur rakhte ho, link bas `https://<username>.github.io/<repo-name>/` jaisa hoga.
- Future mein naye chapters add karne ho to bas nayi `.html` file upload karo aur `index.html` mein ek naya card-link add kar do.
